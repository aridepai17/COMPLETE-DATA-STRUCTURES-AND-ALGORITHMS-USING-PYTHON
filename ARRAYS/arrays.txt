ARRAYS
-> Array can store data of specified type.
-> Elements of array are located in a contiguous manner.
-> Each element of an array has a unique index.

TYPES OF ARRAYS
-> ONE DIMENSIONAL ARRAYS 
    -> An array with a bunch of values having been declared with a single index.
    -> a[i] is i between 0 and n-1.
-> TWO DIMENSIONAL ARRAYS
    -> An array with a bunch of values having been declared with two indexes.
    -> a[i][j] is i between 0 and n-1 and j between 0 and m-1.
-> THREE DIMENSIONAL ARRAYS
    -> An array with a bunch of values having been declared with three indexes.
    -> a[i][j][k] is i between 0 and n-1, j between 0 and m-1 and k between 0 and l-1.

TWO DIMENSIONAL ARRAYS
-> An array with a bunch of values having been declared with double index.
a[i][j] -> i and j between 0 and n
